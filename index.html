Pilot Logbook

Previous Next

The app is a logbook for airclub pilots. The user logs in using a login and password. If the user's status is "pilot", they can view their flights. If their status is "instructor", they can view logs of all pilots, as well as add and delete records.

The program is written in Python3. It uses the MySQL database with relations, views and triggers.

METAR Decoder

Previous Next

The application connects to aviationweather.gov via API and downloads the newest METAR. It's an encoded information about weather conditions in the proximity of over 5000 airports around the world. The user chooses the airport and the program displays the ciphered message. Decode button, tranlates the METAR to simple English text.

The application uses Java and the Swing library.
